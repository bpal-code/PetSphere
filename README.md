# 🐾 PetSphere
### A personal web project for pet care and e-commerce

I built PetSphere to practice building a multi-page website from scratch. It’s a place where users can learn about different pets and "shop" for services. I focused on making it look good on both mobile and desktop while keeping the code clean without using heavy frameworks.

## 🚀 What I Built
- **Species Guides:** Dedicated pages for Dogs, Cats, Birds, Fish, and Rabbits (`dogcare.html`, `rabbit.html`, etc.).
- **Booking System:** A simple interface in `booking.html` for scheduling pet appointments.
- **Vanilla JS Shopping Cart:** I wrote the logic in `cart.js` to handle adding/removing items and calculating totals without using external libraries.
- **Contact Handling:** A `contact.js` script to manage user inputs and form interactions.

## 🚧 Challenges I Overcame
- **The "State" Problem:** It was tricky to keep the shopping cart updated when clicking between different pages (like moving from `bird.html` to `cat.html`). I solved this by refining my JavaScript logic.
- **Consistent Design:** I used modular CSS (`care.css` and `cart.css`) to make sure the colors and buttons looked the same across all 10+ pages.
- **Mobile View:** I spent a lot of time debugging the navigation bar to make sure it didn't break on smaller screens.

## 🛠️ Tech Used
- **HTML5 & CSS3**
- **Vanilla JavaScript** (No frameworks)
- **GitHub Pages** for hosting
